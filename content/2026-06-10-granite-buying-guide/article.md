# Как выбрать гранит и не переплатить: гайд от тех, кто его обрабатывает

## Боль, с которой к нам приходят

80% клиентов не знают разницы между гранитом, габбро и мрамором. Покупают «то что красиво на картинке», получают брак: трещины через год, выцветание на солнце, скол на углу столешницы.

Эта статья — для тех, кто хочет один раз понять что покупает.

## 1. Гранит ≠ гранит. 3 типа, которые путают

**Габбро-диабазит (Карелия)** — чёрный, плотный, не боится мороза. Лучший для памятников. Полируется до зеркала.

**Гранит Возрождение** — серо-розовый, средняя плотность. Универсал: лестницы, цоколь, плитка.

**Лабрадорит** — с синим переливом. Дорогой. Только для декора, не для нагрузки.

Правило: чем темнее — тем плотнее — тем дольше прослужит.

## 2. Почему дешёвый гранит дороже дорогого

Китайский гранит на 30-40% дешевле. Но:
- Толщина 18мм вместо 30мм → скол за 2 года
- Полировка «под лак» → матовеет за сезон
- Нет сертификата радиационной безопасности → штраф при проверке

Реальный кейс клиента: сэкономил 25к на столешнице, через 8 месяцев менял за 70к.

## 3. Как проверить камень перед покупкой

**Тест водой.** Капля воды на необработанной стороне. Впиталась за 30 сек → плохая плотность, будет трескаться.

**Тест ногтем.** Полированная поверхность должна быть гладкой как стекло. Чувствуется текстура — некачественная полировка.

**Сертификат.** Класс 1 — для жилых помещений. Класс 2 — только улица. Без сертификата — не берите.

## 4. Толщина под задачу

| Применение | Толщина |
|------------|---------|
| Памятник | 50-80 мм |
| Столешница | 30 мм |
| Подоконник | 20-30 мм |
| Лестница | 30-40 мм |
| Облицовка | 20 мм |

Меньше — экономия, которая обернётся ремонтом.

## 5. Что входит в честную цену

Камень + распил + полировка + кромка + доставка + монтаж. Если в смете нет одного из пунктов — спросите. Скрытые доплаты — главная жалоба.
